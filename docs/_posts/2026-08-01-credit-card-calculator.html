The formula

Credit card issuers compute your minimum payment as the greater of a flat floor (e.g. $25) or a percentage of the balance. To find the fixed monthly payment that retires a balance in n months, use the standard annuity formula:

PMT = balance * r * (1 + r)^n / ((1 + r)^n - 1)

where r = annual_rate / 12

If you only pay the minimum, most of your payment goes to interest and the balance barely shrinks. The calculator below shows exactly how long payoff takes and how much interest you will pay.

Core calculator

from dataclasses import dataclass


@dataclass
class PayoffRow:
    month: int
    payment: float
    principal: float
    interest: float
    balance: float


def credit_card_schedule(
    balance: float,
    annual_rate: float,
    monthly_payment: float,
    max_months: int = 600,
) -> list[PayoffRow]:
    """Generate a month-by-month payoff schedule.

    Parameters
    ----------
    balance : float
        Current outstanding balance.
    annual_rate : float
        Annual interest rate as a decimal (e.g. 0.22 for 22 %).
    monthly_payment : float
        Fixed amount paid each month.
    max_months : int
        Safety cap to prevent infinite loops on low payments.

    Returns
    -------
    list[PayoffRow]
        One row per month until balance reaches zero.
    """
    r = annual_rate / 12
    schedule: list[PayoffRow] = []

    for month in range(1, max_months + 1):
        interest = round(balance * r, 2)
        payment = min(monthly_payment, balance + interest)
        principal = round(payment - interest, 2)
        balance = round(balance - principal, 2)
        schedule.append(PayoffRow(month, payment, principal, interest, balance))
        if balance <= 0:
            break

    return schedule


# --- quick demo ---
schedule = credit_card_schedule(
    balance=5_000.00,
    annual_rate=0.22,
    monthly_payment=200.00,
)

total_interest = sum(r.interest for r in schedule)
print(f"Months to payoff : {len(schedule)}")
print(f"Total interest   : ${total_interest:,.2f}")
print(f"Total paid       : ${sum(r.payment for r in schedule):,.2f}")

Balance-over-time chart

import plotly.graph_objects as go

schedule = credit_card_schedule(
    balance=5_000.00,
    annual_rate=0.22,
    monthly_payment=200.00,
)

months = [r.month for r in schedule]
balances = [r.balance for r in schedule]
cum_interest = []
running = 0.0
for r in schedule:
    running += r.interest
    cum_interest.append(running)

fig = go.Figure()
fig.add_trace(go.Scatter(
    x=months, y=balances,
    mode="lines", name="Remaining balance",
    line=dict(color="#636EFA", width=2),
))
fig.add_trace(go.Scatter(
    x=months, y=cum_interest,
    mode="lines", name="Cumulative interest",
    line=dict(color="#EF553B", width=2, dash="dot"),
))
fig.update_layout(
    title="Credit Card Payoff: $5,000 at 22 % APR, $200/mo",
    xaxis_title="Month",
    yaxis_title="Dollars ($)",
    template="plotly_dark",
    legend=dict(x=0.55, y=0.95),
    margin=dict(t=50, b=40, l=60, r=20),
)
fig.show()

Key takeaways

  • Doubling your payment from $150 to $300 cuts total interest by more than half.
  • At the minimum payment floor ($25), a $5,000 balance at 22 % takes over 25 years to clear and costs more than $8,000 in interest alone.
  • Use the schedule list to find the exact month where interest drops below principal — that is the tipping point.
