The approach

ESG (Environmental, Social, Governance) scores help investors evaluate companies beyond pure financials. No reliable free API exists for ESG data, so the code below uses a sample dataset that mirrors the structure you would get from MSCI, Sustainalytics, or Bloomberg. The visualizations — radar charts and sector comparisons — work the same with real data.

Radar chart — compare companies

import plotly.graph_objects as go

companies = ["AAPL", "MSFT", "XOM", "JNJ"]
categories = ["Environmental", "Social", "Governance"]

fig = go.Figure()
for company in companies:
    row = esg_data[esg_data["company"] == company].iloc[0]
    values = [row["E"], row["S"], row["G"]]
    # Close the polygon by repeating the first value
    fig.add_trace(go.Scatterpolar(
        r=values + [values[0]],
        theta=categories + [categories[0]],
        name=company,
        fill="toself",
        opacity=0.6,
    ))

fig.update_layout(
    title="ESG Radar Comparison",
    template="plotly_dark",
    polar=dict(
        radialaxis=dict(visible=True, range=[0, 100]),
    ),
    margin=dict(t=60, b=40, l=80, r=80),
)
fig.show()

Sector averages

sector_avg = esg_data.groupby("sector")[["E", "S", "G", "total"]].mean().round(1)
sector_avg = sector_avg.sort_values("total", ascending=True)

fig = go.Figure()
for pillar, color in [("E", "#00CC96"), ("S", "#636EFA"), ("G", "#FFA15A")]:
    fig.add_trace(go.Bar(
        y=sector_avg.index,
        x=sector_avg[pillar],
        name=pillar,
        orientation="h",
        marker_color=color,
    ))

fig.update_layout(
    title="Average ESG Scores by Sector",
    xaxis_title="Score (0-100)",
    template="plotly_dark",
    barmode="group",
    margin=dict(t=50, b=40, l=100, r=20),
    legend=dict(orientation="h", y=1.08, x=0.5, xanchor="center"),
)
fig.show()

Score ranking bar chart

sorted_df = esg_data.sort_values("total", ascending=True)

fig = go.Figure(go.Bar(
    y=sorted_df["company"],
    x=sorted_df["total"],
    orientation="h",
    marker_color=[
        "#00CC96" if t >= 75 else "#FFA15A" if t >= 60 else "#EF553B"
        for t in sorted_df["total"]
    ],
    text=[f"{t}" for t in sorted_df["total"]],
    textposition="outside",
))
fig.update_layout(
    title="ESG Total Score Ranking",
    xaxis_title="Total Score (0-100)",
    template="plotly_dark",
    margin=dict(t=50, b=40, l=80, r=40),
    xaxis=dict(range=[0, 100]),
)
fig.show()

Key takeaways

  • Radar charts expose the shape of a company's ESG profile. A company might score high overall but have a single weak pillar — the radar makes that obvious.
  • Sector comparisons reveal structural biases: energy companies tend to score low on E by nature, while utilities with renewables can score very high.
  • ESG scores vary dramatically across providers. MSCI, Sustainalytics, and S&P often disagree on the same company. Always check which provider your data comes from.
  • The heatmap is the fastest way to scan many companies at once — red cells jump out as areas needing attention.
  • Color-coding the ranking bar (green ≥ 75, orange ≥ 60, red < 60) gives an instant pass/watch/fail visual filter for screening.
