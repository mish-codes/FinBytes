The concept

Combine price data with sentiment scores from news headlines to build a dashboard that shows whether market mood and price action are aligned. When sentiment diverges from price (e.g., headlines turn negative while prices rise), it can signal a potential reversal.

VADER sentiment scoring

from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer

analyzer = SentimentIntensityAnalyzer()

def score_headline(text: str) -> dict:
    scores = analyzer.polarity_scores(text)
    return {
        "compound": scores["compound"],
        "pos": scores["pos"],
        "neg": scores["neg"],
        "neu": scores["neu"],
    }

sentiment_scores = headlines_df["headline"].apply(lambda h: pd.Series(score_headline(h)))
headlines_df = pd.concat([headlines_df, sentiment_scores], axis=1)

# Classify
headlines_df["label"] = "Neutral"
headlines_df.loc[headlines_df["compound"] >= 0.05, "label"] = "Positive"
headlines_df.loc[headlines_df["compound"] <= -0.05, "label"] = "Negative"

print(headlines_df[["date", "compound", "label", "headline"]].to_string(index=False))

Synthetic price data (simulating yfinance)

# In production, replace this block with:

np.random.seed(42)

# Generate prices for the same date range
biz_dates = pd.bdate_range("2025-01-06", "2025-02-03")
returns = np.random.normal(0.0004, 0.01, len(biz_dates))

# Correlate loosely with sentiment
daily_sentiment = headlines_df.set_index("date")["compound"].reindex(biz_dates).fillna(0)
returns += daily_sentiment.values * 0.005  # sentiment has a small effect

prices = 580 * np.cumprod(1 + returns)  # SPY-like starting price
price_df = pd.DataFrame({"date": biz_dates, "price": prices}).set_index("date")

print(f"Price range: ${price_df['price'].min():.2f} - ${price_df['price'].max():.2f}")

Merge price and sentiment

daily = price_df.copy()
daily["sentiment"] = headlines_df.set_index("date")["compound"].reindex(daily.index)
daily["sentiment"] = daily["sentiment"].fillna(method="ffill").fillna(0)

# Rolling averages
daily["sentiment_3d"] = daily["sentiment"].rolling(3, min_periods=1).mean()
daily["price_return"] = daily["price"].pct_change()

print(daily.head(10))

Dual-axis chart: price + sentiment

import plotly.graph_objects as go

fig.update_yaxes(title_text="Price ($)", secondary_y=False)

Correlation analysis

from scipy import stats

# Contemporaneous correlation: same-day sentiment vs return
valid = daily.dropna(subset=["price_return", "sentiment"])
corr_same = valid["sentiment"].corr(valid["price_return"])

# Lagged: today's sentiment vs tomorrow's return
valid["next_return"] = valid["price_return"].shift(-1)
valid_lagged = valid.dropna(subset=["next_return"])
corr_lagged = valid_lagged["sentiment"].corr(valid_lagged["next_return"])

print(f"Same-day correlation (sentiment vs return):  {corr_same:.4f}")
print(f"Next-day correlation (sentiment vs return):  {corr_lagged:.4f}")

# Statistical significance
if len(valid_lagged) > 5:
    r, p = stats.pearsonr(valid_lagged["sentiment"], valid_lagged["next_return"])
    print(f"Lagged correlation p-value: {p:.4f} {'(significant)' if p < 0.05 else '(not significant)'}")

Rolling sentiment with divergence detection

daily["price_sma5"] = daily["price"].rolling(5, min_periods=1).mean()
daily["price_above_sma"] = daily["price"] > daily["price_sma5"]
daily["sentiment_positive"] = daily["sentiment_3d"] > 0

# Divergence: price rising but sentiment negative, or vice versa
daily["divergence"] = daily["price_above_sma"] != daily["sentiment_positive"]

divergence_dates = daily[daily["divergence"]].index
print(f"Divergence days: {len(divergence_dates)} out of {len(daily)}")

fig = go.Figure()
fig.add_trace(go.Scatter(x=daily.index, y=daily["price"], mode="lines",
                         line=dict(color="steelblue"), name="Price"))

if len(divergence_dates) > 0:
    fig.add_trace(go.Scatter(
        x=divergence_dates, y=daily.loc[divergence_dates, "price"],
        mode="markers", marker=dict(color="orange", size=10, symbol="diamond"),
        name="Divergence",
    ))

fig.update_layout(title="Price-Sentiment Divergence",
                  xaxis_title="Date", yaxis_title="Price",
                  height=400, width=800, template="plotly_white")
fig.show()

For real headlines, use a news API (NewsAPI, Alpha Vantage, or scrape RSS feeds). The sentiment pipeline stays the same. The key insight is that VADER works best on short, punchy text like headlines. For longer articles, consider FinBERT or a transformer-based model fine-tuned on financial text.

Limitations

  • Small sample — 20 headlines is too few for reliable correlation. Production systems process thousands of headlines per day.
  • Causality — correlation does not imply causation. Headlines often react to prices, not the other way around.
  • Sentiment is noisy — a single headline can be misleading. Aggregate across many sources before making decisions.
