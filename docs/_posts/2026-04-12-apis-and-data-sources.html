How Each Source Works

yfinance — Market Data

The most-used data source across the dashboard. A Python wrapper around Yahoo Finance that returns pandas DataFrames. No API key, no signup, no rate limiting for reasonable usage.

import yfinance as yf

df = yf.download("AAPL", period="1y", auto_adjust=True, progress=False)

Returns OHLCV (Open, High, Low, Close, Volume) indexed by date. Used by 12+ dashboard pages for price history, technical indicators, returns, and risk metrics.

CoinGecko — Crypto Prices

Free REST API for cryptocurrency prices. Returns current price, 24-hour change, and market cap in any fiat currency. No key needed for the simple price endpoint.

url = "https://api.coingecko.com/api/v3/simple/price"
params = {"ids": "bitcoin,ethereum", "vs_currencies": "usd",
          "include_24hr_change": "true", "include_market_cap": "true"}
r = requests.get(url, params=params, timeout=10)

Open Exchange Rates — FX

Free tier covers 150+ currencies against USD. Returns a flat JSON object of rates. No signup required for the open endpoint.

r = requests.get("https://open.er-api.com/v6/latest/USD", timeout=10)
rates = r.json()["rates"]  # {"EUR": 0.92, "GBP": 0.79, ...}

Central Bank APIs — Benchmark Rates

Three central banks, three different API formats, same data type (overnight interest rate):

  • Bank of England — CSV download with a custom URL scheme. SONIA rate series code IUDSNPY.
  • ECB — REST API returning CSV. Euro Short-Term Rate (€STR) via the EST flow.
  • New York Fed — REST API returning CSV. SOFR from the markets data service.

All three are free, require no authentication, and return date/rate pairs.

HM Land Registry — Property Prices

The definitive UK property price dataset. Published monthly as CSV files at predictable URLs. Each year is ~150MB with every transaction in England and Wales. We download 10 years, filter to London postcodes, and compress to a 5.6MB Parquet file.

url = f"https://price-paid-data.publicdata.landregistry.gov.uk/pp-{year}.csv"
df = pd.read_csv(url, header=None, names=PPD_COLS)

OpenStreetMap Overpass — Spatial Queries

Queries the full OpenStreetMap database. We use it to find brand shop locations in London for the “Waitrose effect” analysis. Free, no auth, but rate-limited — retry with backoff on 429 responses.

[out:json][timeout:60][bbox:51.28,-0.51,51.69,0.33];

GitHub Actions API — CI Status

The System Health page checks recent CI runs to show a green/red status. Public repos need no authentication for the /actions/runs endpoint.

r = requests.get(
    f"https://api.github.com/repos/{REPO}/actions/runs",
    params={"per_page": 3}, timeout=10,
)

Common Patterns

  • All sources are free and require no API key. The only authenticated endpoint is the internal Scanner API on Render.
  • Every external call is wrapped in try/except with a fallback (cached data, sample data, or a user-friendly error message).
  • Streamlit caching (@st.cache_data) prevents redundant API calls on every rerun.
  • Tests mock all external calls via conftest.py autouse fixtures — no test ever hits the network.
