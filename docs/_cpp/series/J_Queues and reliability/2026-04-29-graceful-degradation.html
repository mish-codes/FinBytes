Fail safe, not fail hard

When a dependency fails, the choice is not just work normally vs crash. A third option: degrade gracefully. Continue operating with reduced functionality, use stale data when fresh data is unavailable, and be transparent about what mode you are in.

Degradation strategies:

  • Stale pricing: if the feed is unavailable, use last known prices with an age indicator.
  • Model fallback: if complex vol surface calibration fails, fall back to flat vol. Report which model was used.
  • Read-only mode: if the booking system is unavailable, allow read (risk/position queries) but reject new trades.
  • Cached results: if a pricer is overloaded, return cached results for repeated queries until capacity recovers.

The key rule: always be transparent about degraded state. A system that silently returns stale data is dangerous. One that logs "using stale price from 45s ago" gives the trader context to decide whether to act on it.

Python graceful degradation

import time, logging
from dataclasses import dataclass
from typing import Optional

@dataclass
class PriceResult:
    price:    float
    symbol:   str
    age_s:    float
    is_stale: bool
    source:   str  # "live", "cached", "stale_cache"

class ResilientPricer:
    STALE_THRESHOLD_S = 60.0

    def __init__(self, live_feed, fallback_feed=None):
        self._live     = live_feed
        self._fallback = fallback_feed
        self._cache:   dict[str, tuple[float, float]] = {}  # sym -> (price, ts)

    def get_price(self, symbol: str) -> PriceResult:
        try:
            price = self._live.get_price(symbol)
            self._cache[symbol] = (price, time.monotonic())
            return PriceResult(price, symbol, 0.0, False, "live")
        except Exception as e:
            logging.warning(f"Live feed failed for {symbol}: {e}")

        if symbol in self._cache:
            cached_price, ts = self._cache[symbol]
            age = time.monotonic() - ts
            stale = age > self.STALE_THRESHOLD_S
            if stale: logging.warning(f"Stale price for {symbol}: {age:.0f}s old")
            return PriceResult(cached_price, symbol, age, stale,
                               "stale_cache" if stale else "cached")

        raise RuntimeError(f"No price available for {symbol}")

C# graceful degradation

public record PriceResult(double Price, string Symbol,
    double AgeSeconds, bool IsStale, string Source);

public class ResilientPricer {
    private readonly IMarketFeed _live;
    private readonly Dictionary<string, (double Price, DateTime Ts)> _cache = new();
    private const double StaleSec = 60.0;

    public ResilientPricer(IMarketFeed live) => _live = live;

    public PriceResult GetPrice(string symbol) {
        try {
            var price = _live.GetPrice(symbol);
            _cache[symbol] = (price, DateTime.UtcNow);
            return new(price, symbol, 0, false, "live");
        } catch (Exception ex) {
            Log.Warning(ex, "Live feed failed for {Symbol}", symbol);
        }
        if (_cache.TryGetValue(symbol, out var cached)) {
            var age   = (DateTime.UtcNow - cached.Ts).TotalSeconds;
            var stale = age > StaleSec;
            if (stale) Log.Warning("Stale price for {Symbol}: {Age:F0}s", symbol, age);
            return new(cached.Price, symbol, age, stale, stale ? "stale_cache" : "cached");
        }
        throw new InvalidOperationException($"No price available for {symbol}");
    }
}

C++ graceful degradation

#include <string>
#include <unordered_map>
#include <chrono>
#include <optional>
#include <stdexcept>

struct PriceResult {
    double price; std::string symbol; double age_s; bool is_stale; std::string source;
};

class ResilientPricer {
    static constexpr double STALE_S = 60.0;
    struct CacheEntry { double price; double ts; };
    std::unordered_map<std::string, CacheEntry> cache_;

    double now_s() const {
        return std::chrono::duration<double>(
            std::chrono::steady_clock::now().time_since_epoch()).count();
    }
public:
    PriceResult get_price(const std::string& sym) {
        try {
            double price = live_feed_.get_price(sym);
            cache_[sym] = {price, now_s()};
            return {price, sym, 0.0, false, "live"};
        } catch (...) {}

        auto it = cache_.find(sym);
        if (it != cache_.end()) {
            double age   = now_s() - it->second.ts;
            bool   stale = age > STALE_S;
            return {it->second.price, sym, age, stale, stale ? "stale_cache" : "cached"};
        }
        throw std::runtime_error("No price available: " + sym);
    }
};
Always include source and age in your price result. Downstream consumers - risk engines, traders - need to know whether they are acting on live or stale data. A price without provenance is an untrusted number.

Rust graceful degradation

use std::collections::HashMap;
use std::sync::Arc;
use std::time::{Duration, Instant};
use tokio::sync::Mutex;

#[derive(Clone, Debug)]
struct PriceResult {
    price:    f64,
    symbol:   String,
    age_s:    f64,
    is_stale: bool,
    source:   String, // "live", "cached", "stale_cache"
}

struct CacheEntry {
    price: f64,
    ts:    Instant,
}

struct ResilientPricer {
    cache:           Arc<Mutex<HashMap<String, CacheEntry>>>,
    stale_threshold: Duration,
}

impl ResilientPricer {
    fn new(stale_threshold_s: f64) -> Self {
        Self {
            cache: Arc::new(Mutex::new(HashMap::new())),
            stale_threshold: Duration::from_secs_f64(stale_threshold_s),
        }
    }

    async fn get_price(&self, symbol: &str) -> Result<PriceResult, String> {
        // Try live feed first
        match live_feed_get_price(symbol).await {
            Ok(price) => {
                let mut cache = self.cache.lock().await;
                cache.insert(symbol.to_string(), CacheEntry { price, ts: Instant::now() });
                return Ok(PriceResult {
                    price, symbol: symbol.into(), age_s: 0.0,
                    is_stale: false, source: "live".into(),
                });
            }
            Err(e) => eprintln!("Live feed failed for {}: {}", symbol, e),
        }

        // Fall back to cache
        let cache = self.cache.lock().await;
        if let Some(entry) = cache.get(symbol) {
            let age = entry.ts.elapsed().as_secs_f64();
            let is_stale = age > self.stale_threshold.as_secs_f64();
            if is_stale {
                eprintln!("WARN: stale price for {}: {:.0}s old", symbol, age);
            }
            return Ok(PriceResult {
                price: entry.price, symbol: symbol.into(), age_s: age,
                is_stale,
                source: if is_stale { "stale_cache" } else { "cached" }.into(),
            });
        }

        Err(format!("No price available for {}", symbol))
    }
}

// Model fallback — try complex vol surface, fall back to flat vol
async fn price_with_fallback(trade: &Trade) -> PriceResult {
    match price_with_vol_surface(trade).await {
        Ok(result) => result,
        Err(e) => {
            eprintln!("Vol surface failed: {} — using flat vol", e);
            price_with_flat_vol(trade) // simpler model, always succeeds
        }
    }
}
Rust's Result type makes the fallback chain explicit — each step either succeeds or falls through to the next. The compiler ensures every error path is handled. The source field in PriceResult propagates provenance to downstream consumers.

Go graceful degradation

package main

import (
    "fmt"
    "log"
    "sync"
    "time"
)

type PriceResult struct {
    Price   float64
    Symbol  string
    AgeS    float64
    IsStale bool
    Source  string // "live", "cached", "stale_cache"
}

type cacheEntry struct {
    price float64
    ts    time.Time
}

type ResilientPricer struct {
    mu             sync.RWMutex
    cache          map[string]cacheEntry
    staleThreshold time.Duration
    liveFeed       MarketFeed
}

func NewResilientPricer(feed MarketFeed, staleSec float64) *ResilientPricer {
    return &ResilientPricer{
        cache:          make(map[string]cacheEntry),
        staleThreshold: time.Duration(staleSec * float64(time.Second)),
        liveFeed:       feed,
    }
}

func (rp *ResilientPricer) GetPrice(symbol string) (PriceResult, error) {
    // Try live feed
    price, err := rp.liveFeed.GetPrice(symbol)
    if err == nil {
        rp.mu.Lock()
        rp.cache[symbol] = cacheEntry{price, time.Now()}
        rp.mu.Unlock()
        return PriceResult{price, symbol, 0, false, "live"}, nil
    }
    log.Printf("Live feed failed for %s: %v", symbol, err)

    // Fall back to cache
    rp.mu.RLock()
    entry, ok := rp.cache[symbol]
    rp.mu.RUnlock()
    if ok {
        age := time.Since(entry.ts).Seconds()
        stale := age > rp.staleThreshold.Seconds()
        if stale {
            log.Printf("WARN: stale price for %s: %.0fs old", symbol, age)
        }
        source := "cached"
        if stale { source = "stale_cache" }
        return PriceResult{entry.price, symbol, age, stale, source}, nil
    }

    return PriceResult{}, fmt.Errorf("no price available for %s", symbol)
}

// Read-only mode — context signals degraded state
type ServiceMode int

const (
    ModeNormal ServiceMode = iota
    ModeReadOnly
    ModeDegraded
)

type DegradableService struct {
    mu   sync.RWMutex
    mode ServiceMode
}

func (s *DegradableService) SetMode(m ServiceMode) {
    s.mu.Lock()
    defer s.mu.Unlock()
    if s.mode != m {
        log.Printf("Service mode: %v -> %v", s.mode, m)
    }
    s.mode = m
}

func (s *DegradableService) SubmitTrade(trade Trade) error {
    s.mu.RLock()
    defer s.mu.RUnlock()
    if s.mode == ModeReadOnly {
        return fmt.Errorf("service in read-only mode — trade submission rejected")
    }
    return processTrade(trade)
}
Go's sync.RWMutex lets multiple readers (price queries) run concurrently while locking only for writes (cache updates). The ServiceMode pattern with a simple int and a mutex is idiomatic Go for managing degraded state — log every transition so operators can see exactly when the system entered and exited degraded mode.

TypeScript graceful degradation

interface PriceResult {
  price:   number;
  symbol:  string;
  ageS:    number;
  isStale: boolean;
  source:  "live" | "cached" | "stale_cache";
}

class ResilientPricer {
  private cache = new Map<string, { price: number; ts: number }>();
  private static STALE_MS = 60_000;

  constructor(private liveFeed: MarketFeed) {}

  async getPrice(symbol: string): Promise<PriceResult> {
    // Try live feed
    try {
      const price = await this.liveFeed.getPrice(symbol);
      this.cache.set(symbol, { price, ts: Date.now() });
      return { price, symbol, ageS: 0, isStale: false, source: "live" };
    } catch (err) {
      console.warn(`Live feed failed for ${symbol}: ${err}`);
    }

    // Fall back to cache
    const cached = this.cache.get(symbol);
    if (cached) {
      const ageMs = Date.now() - cached.ts;
      const isStale = ageMs > ResilientPricer.STALE_MS;
      if (isStale) console.warn(`Stale price for ${symbol}: ${(ageMs/1000).toFixed(0)}s old`);
      return {
        price: cached.price, symbol,
        ageS: ageMs / 1000, isStale,
        source: isStale ? "stale_cache" : "cached",
      };
    }

    throw new Error(`No price available for ${symbol}`);
  }
}

// Model fallback — try complex model, fall back to simple
async function priceWithFallback(trade: Trade): Promise<PriceResult> {
  try {
    return await priceWithVolSurface(trade);
  } catch (err) {
    console.warn(`Vol surface failed: ${err} — using flat vol`);
    return priceWithFlatVol(trade);
  }
}

// Service mode management
type ServiceMode = "normal" | "read-only" | "degraded";

class DegradableService {
  private mode: ServiceMode = "normal";

  setMode(mode: ServiceMode): void {
    if (this.mode !== mode) {
      console.log(`Service mode: ${this.mode} -> ${mode}`);
      this.mode = mode;
    }
  }

  async submitTrade(trade: Trade): Promise<void> {
    if (this.mode === "read-only") {
      throw new Error("Service in read-only mode — trade submission rejected");
    }
    await processTrade(trade);
  }

  get currentMode(): ServiceMode { return this.mode; }
}
TypeScript's union type "live" | "cached" | "stale_cache" for the source field gives you compile-time safety that the source string is always one of the expected values. The ServiceMode union type similarly constrains the degradation states.

Key observations

  • Rust's Result chain makes fallback logic the clearest. Each step returns Result<T, E>, and the compiler forces you to handle failures. You cannot accidentally skip a fallback level. The source field with typed enums ensures provenance is always correct.
  • Go's sync.RWMutex makes concurrent cache access trivial. Multiple price readers run concurrently; only cache updates take a write lock. Combined with goroutines for mode switching and monitoring, Go makes degradation management feel lightweight.
  • TypeScript gets concurrent cache access for free because Node.js is single-threaded — no lock needed. The tradeoff: you cannot run CPU-intensive fallback logic (like flat vol pricing) without blocking the event loop.
  • C# has the richest ecosystem for degradation patterns — Polly fallback policies, ASP.NET health checks, and feature flags (LaunchDarkly, Azure Feature Management) work together out of the box.
  • Always log the source of every price. In all languages, the source field is not optional. A risk engine that silently uses stale data without knowing it is a compliance and P&L risk.

Decision guide for graceful degradation

  • Compiler-enforced fallback handling: Rust — Result chains make skipping a fallback a compile error
  • Concurrent read-heavy workload (many symbols): Go with RWMutex or TypeScript (single-threaded, lock-free)
  • Enterprise with feature flags: C# — richest integration with feature management platforms
  • Always include provenance. Every price result must carry source and age — downstream consumers need this to make informed decisions
  • Log every mode transition. When the system enters or exits degraded mode, log it prominently so operators and audit trails capture the event
