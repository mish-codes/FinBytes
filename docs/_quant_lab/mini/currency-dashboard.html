What it does

Fetches live spot rates from a free exchange rate API, builds a cross-rate matrix, converts between any currency pair, and plots historical forex data. A complete currency monitoring tool.

The math

Conversion uses triangulation through a base currency: amount_dst = (amount / rate_src) * rate_dst. The cross-rate matrix shows every pair at a glance. Normalizing historical data to percentage change lets you compare pairs with very different scales on the same chart.

When to use it

Use this for monitoring forex exposure, travel planning, or spotting arbitrage opportunities. If A->B->C->A yields more than 1.0 in the cross-rate matrix, something is mispriced.

Core function

import requests
from dataclasses import dataclass


@dataclass
class CurrencyConverter:
    """Convert between currencies using live rates."""
    base: str = "USD"
    rates: dict[str, float] | None = None

    def fetch_rates(self) -> None:
        url = f"https://api.exchangerate.host/latest?base={self.base}"
        resp = requests.get(url, timeout=10)
        resp.raise_for_status()
        self.rates = resp.json()["rates"]

    def convert(self, amount: float, src: str, dst: str) -> float:
        if self.rates is None:
            self.fetch_rates()
        in_base = amount / self.rates[src]
        return round(in_base * self.rates[dst], 4)

    def rate(self, src: str, dst: str) -> float:
        return self.convert(1.0, src, dst)

Quick usage

fx = CurrencyConverter(base="USD")
fx.fetch_rates()
print(f"1 EUR = {fx.rate('EUR', 'USD'):.4f} USD")
print(f"1 GBP = {fx.rate('GBP', 'JPY'):.2f} JPY")
