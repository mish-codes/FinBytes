What it does

Fetches live crypto prices from CoinGecko's free API (no key required), values your holdings in USD, computes allocation percentages, and visualizes the portfolio with a donut chart and stacked area chart over time.

The math

Valuation is straightforward: value = quantity * price. Allocation = value / total_value * 100. The stacked area chart uses yfinance crypto tickers (e.g., BTC-USD) to show historical portfolio value with each coin's contribution visible.

When to use it

Use this to monitor crypto exposure and rebalance. CoinGecko's /simple/price endpoint is free with a rate limit of ~10-30 calls/minute. Crypto trades 24/7, so there are no market-hours gaps in the data.

Core function

import requests
from dataclasses import dataclass, field


def fetch_crypto_prices(coin_ids: list[str], currency: str = "usd") -> dict:
    """Fetch price, 24h change, and market cap from CoinGecko."""
    url = "https://api.coingecko.com/api/v3/simple/price"
    params = {
        "ids": ",".join(coin_ids),
        "vs_currencies": currency,
        "include_24hr_change": "true",
        "include_market_cap": "true",
    }
    resp = requests.get(url, params=params, timeout=10)
    resp.raise_for_status()
    return resp.json()


@dataclass
class CryptoPortfolio:
    holdings: dict[str, float] = field(default_factory=dict)
    prices: dict[str, dict] | None = None

    def refresh_prices(self) -> None:
        self.prices = fetch_crypto_prices(list(self.holdings.keys()))

    def valuations(self) -> dict[str, float]:
        if self.prices is None:
            self.refresh_prices()
        return {
            coin: qty * self.prices[coin]["usd"]
            for coin, qty in self.holdings.items()
        }

    @property
    def total_value(self) -> float:
        return sum(self.valuations().values())

Quick usage

portfolio = CryptoPortfolio(holdings={
    "bitcoin": 0.5, "ethereum": 4.0, "solana": 25.0,
})
portfolio.refresh_prices()
print(f"Total value: ${portfolio.total_value:,.2f}")
