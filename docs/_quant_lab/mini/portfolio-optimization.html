What it does

Computes the efficient frontier using Markowitz portfolio theory. Finds the maximum Sharpe ratio portfolio and minimum variance portfolio via scipy optimization. Visualizes thousands of random portfolios colored by Sharpe ratio with the frontier overlaid.

The math

Portfolio return = w' * mu * 252. Portfolio volatility = sqrt(w' * Cov * w) * sqrt(252). Sharpe = (return - risk_free) / volatility. The efficient frontier minimizes volatility at each target return level, subject to weights summing to 1 with long-only bounds.

When to use it

Use this for asset allocation decisions. Caveat: Markowitz is notoriously sensitive to input estimates. Small changes in expected returns produce large weight swings. In practice, use shrinkage estimators (Ledoit-Wolf) or Black-Litterman for more robust results.

Core function

import numpy as np
from scipy.optimize import minimize


def portfolio_performance(
    weights: np.ndarray,
    mean_returns: np.ndarray,
    cov_matrix: np.ndarray,
    risk_free_rate: float = 0.04,
) -> tuple[float, float, float]:
    """Return annualized return, volatility, and Sharpe ratio."""
    ret = np.sum(mean_returns * weights) * 252
    vol = np.sqrt(weights @ cov_matrix @ weights) * np.sqrt(252)
    sharpe = (ret - risk_free_rate) / vol
    return ret, vol, sharpe


def optimize_portfolio(mean_returns, cov_matrix, n_assets, rf=0.04):
    """Find max-Sharpe and min-variance portfolios."""
    bounds = tuple((0, 1) for _ in range(n_assets))
    constraints = {"type": "eq", "fun": lambda w: np.sum(w) - 1}
    init = np.ones(n_assets) / n_assets

    # Max Sharpe
    neg_sharpe = lambda w: -portfolio_performance(w, mean_returns, cov_matrix, rf)[2]
    ms = minimize(neg_sharpe, init, method="SLSQP", bounds=bounds, constraints=constraints)

    # Min variance
    vol_fn = lambda w: np.sqrt(w @ cov_matrix @ w) * np.sqrt(252)
    mv = minimize(vol_fn, init, method="SLSQP", bounds=bounds, constraints=constraints)

    return ms.x, mv.x

Quick usage

ms_weights, mv_weights = optimize_portfolio(mean_returns, cov_matrix, 5)
ret, vol, sharpe = portfolio_performance(ms_weights, mean_returns, cov_matrix)
