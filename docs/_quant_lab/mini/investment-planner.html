What it does

Computes month-by-month investment growth for a lump sum plus regular contributions. Includes a what-if mode to compare different annual return assumptions side by side.

The math

FV = P * (1 + r)^n  +  C * ((1 + r)^n - 1) / r

P = initial principal, C = monthly contribution
r = monthly rate, n = total months

The first term is growth on the initial deposit. The second is the annuity -- growth on the stream of contributions.

When to use it

Use this to visualize how compounding works over time. At 8% over 20 years, growth exceeds contributions. The gap between 5% and 10% return is $176,000 on the same contributions, showing how sensitive outcomes are to return assumptions.

Core function

from dataclasses import dataclass


@dataclass
class GrowthRow:
    month: int
    contribution_total: float
    growth: float
    balance: float


def investment_growth(
    principal: float,
    monthly_contribution: float,
    annual_rate: float,
    years: int,
) -> list[GrowthRow]:
    """Month-by-month investment growth schedule."""
    r = annual_rate / 12
    balance = principal
    contributions = principal
    schedule: list[GrowthRow] = []

    for month in range(1, years * 12 + 1):
        balance = balance * (1 + r) + monthly_contribution
        contributions += monthly_contribution
        growth = round(balance - contributions, 2)
        schedule.append(GrowthRow(month, round(contributions, 2),
                                  growth, round(balance, 2)))

    return schedule

Quick usage

rows = investment_growth(
    principal=10_000, monthly_contribution=500,
    annual_rate=0.08, years=20,
)
final = rows[-1]
print(f"Total contributed: ${final.contribution_total:,.2f}")
print(f"Final balance:     ${final.balance:,.2f}")
