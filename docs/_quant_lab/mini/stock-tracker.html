What it does

Fetches stock data via yfinance and builds an interactive candlestick chart with volume bars, a key metrics snapshot (price, market cap, P/E, 52-week range), and a gauge chart showing where the current price sits within its 52-week range.

The math

Candlestick charts encode open, high, low, close in a single bar. Volume bars are color-coded by candle direction (green for up, red for down) to show conviction behind price moves. The 52-week gauge provides an instant read on relative valuation.

When to use it

Use this for daily monitoring of a watchlist. The gauge chart shows whether a stock is near its 52-week low (potential value) or high (potential overextension). Wrap the snapshot class in a loop to build a full dashboard table programmatically.

Core function

import yfinance as yf
from dataclasses import dataclass


@dataclass
class StockSnapshot:
    ticker: str
    price: float
    change_pct: float
    market_cap: str
    pe_ratio: float | None
    week52_low: float
    week52_high: float
    volume: int

    @classmethod
    def from_yfinance(cls, ticker: str) -> "StockSnapshot":
        t = yf.Ticker(ticker)
        info = t.info
        prev = info.get("previousClose", 1)
        price = info.get("currentPrice", info.get("regularMarketPrice", 0))
        cap = info.get("marketCap", 0)
        cap_str = f"${cap/1e12:.2f}T" if cap >= 1e12 else f"${cap/1e9:.2f}B"
        return cls(
            ticker=ticker, price=price,
            change_pct=round((price / prev - 1) * 100, 2),
            market_cap=cap_str,
            pe_ratio=info.get("trailingPE"),
            week52_low=info.get("fiftyTwoWeekLow", 0),
            week52_high=info.get("fiftyTwoWeekHigh", 0),
            volume=info.get("volume", 0),
        )

Quick usage

snap = StockSnapshot.from_yfinance("AAPL")
print(f"{snap.ticker}: ${snap.price:.2f} ({snap.change_pct:+.2f}%)")
print(f"52-week: ${snap.week52_low:.2f} - ${snap.week52_high:.2f}")
