What it does

Replays two geopolitical shock events — the 2020 US–Iran escalation and the 2024–2026 Strait of Hormuz tensions — month by month on a 3D globe. Each frame shows how tightly five destination markets (India, Turkey, Germany, US, UK) were moving with a composite Middle East risk index at that moment in time.

Arc colour encodes the direction of contagion: red arcs mean the destination market was co-moving with the shock (risk-on spillover), green means decoupling or safe-haven bid, amber is a weak or ambiguous signal, and slate is noise. Country fills on the globe use the same colour ramp so you can read the correlation field at a glance without focusing on individual arcs.

A side panel tracks four stress-signal instruments — Brent Crude, Baltic Dry shipping, Gold, and VIX — with sparklines that advance in sync with the globe, so you can see the transmission chain: Middle East shock → oil spike → shipping rerouting → safe-haven bid → equity fear.

The math

Middle East risk index: simple daily mean of three epicenter-region ETF closes (Israel EIS, Saudi Arabia KSA, UAE UAE). Bond-yield series for those sovereigns aren't cleanly available via free data, so equity ETFs proxy the same risk premium. Their daily mean becomes the ME index used as the left-hand side of every correlation.

Rolling correlation: 7-trading-day Pearson r between the ME index and each destination asset. Pearson r ∈ [−1, +1] maps to the diverging colour ramp. A 7-day window is short enough to show how correlations shift within a single week of headlines, while being long enough to avoid single-day noise dominating the signal.

corr = s1.rolling(window=7).corr(s2)  # aligned daily return series

Two correlation regimes: FRED monthly yield series (India 10Y, Germany 10Y) cannot be mixed into a 7-day window — forward-filling monthly data onto a daily calendar produces zero-variance stretches that blow up to ±∞ correlations. Those tickers use a 3-month rolling window on the monthly cadence instead. Values update once per month rather than daily, but the numbers are real.

The stress-signal panel uses polarity-aware colouring: rising Brent, shipping rates, or VIX is red (stress-on); rising Gold is green (safe-haven bid working). All values are measured as cumulative % change from the period start, with an amber band at ±2% treated as noise rather than signal.

When to use it

Use this to build intuition about how geopolitical shocks propagate — which markets decouple, which get pulled in, and how quickly the correlation signal decays after the acute phase. Not a trading signal: rolling 7-day correlations are noisy and highly sensitive to window choice. Treat the arcs as qualitative direction, not quantitative edge.

Middle East risk index

import pandas as pd


def middle_east_index(events: pd.DataFrame) -> pd.Series:
    """Daily mean of epicenter-tagged closing values.

    events columns: date, ticker, asset_role, close
    Returns a Series indexed by date.
    """
    epi = events[events["asset_role"] == "epicenter"]
    wide = epi.pivot_table(index="date", columns="ticker", values="close")
    return wide.mean(axis=1)

Rolling correlation per destination

def correlations_for_date(
    events: pd.DataFrame,
    selected_date: date,
) -> dict[str, float]:
    """Pearson r between ME index and each destination on selected_date.

    Daily tickers use a 7-day window; FRED monthly tickers use 3 months.
    Mixing monthly data into a 7-day daily window produces zero-variance
    stretches → ±inf correlations, so each ticker gets the window that
    matches its native cadence.
    """
    me_index = middle_east_index(events)
    result = {}
    for country_code, meta in DESTINATION_CITIES.items():
        ticker = meta["ticker"]
        window = 3 if ticker.startswith("FRED:") else 7
        dest = events[events["ticker"] == ticker].set_index("date")["close"]
        aligned = pd.concat([me_index, dest], axis=1, join="inner").dropna()
        corr_series = (
            aligned.iloc[:, 0]
            .rolling(window)
            .corr(aligned.iloc[:, 1])
            .replace([float("inf"), float("-inf")], float("nan"))
            .dropna()
        )
        mask = corr_series.index <= pd.Timestamp(selected_date).date()
        result[country_code] = float(corr_series[mask].iloc[-1]) if mask.any() else 0.0
    return result

Arc colour ramp

def corr_arc_color(v: float, alpha: float) -> str:
    """Diverging rgba string: green (decoupling) → slate (noise) → red (contagion).

    Thresholds:
      |v| >= 0.5  → strong signal  (red or green)
      |v| >= 0.2  → weak signal    (amber)
      |v| <  0.2  → noise          (slate)
    """
    if v >= 0.5:
        return f"rgba(180,70,70,{alpha:.2f})"    # muted rose — contagion
    if v <= -0.5:
        return f"rgba(50,140,80,{alpha:.2f})"    # muted sage — decoupling
    if abs(v) >= 0.2:
        return f"rgba(200,140,40,{alpha:.2f})"   # amber — weak signal
    return f"rgba(71,85,105,{alpha:.2f})"        # #475569 slate — noise
