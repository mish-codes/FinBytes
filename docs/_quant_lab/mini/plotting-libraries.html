What it does

Fetches OHLCV data for any ticker and renders four chart types (line, candlestick, volume bar, returns histogram) using four different Python plotting libraries. The data table is editable — spike a value to see how each library handles outliers.

Why four libraries

Each library represents a different philosophy: Plotly is interactive and declarative, Matplotlib is imperative with pixel control, Altair uses grammar-of-graphics for concise specs, and Bokeh offers server-side rendering with glyph composition. Comparing them side-by-side reveals which tool fits which job.

Outlier handling

Financial data regularly contains extreme values. This project flags values beyond 3 standard deviations and demonstrates each library's approach to marking outliers: annotations (Plotly), arrow labels (Matplotlib), conditional encoding (Altair), and glyph overlays (Bokeh).

Outlier detection

def detect_outliers(df, n_std=3.0):
    numeric = df.select_dtypes(include=[np.number])
    z_scores = (numeric - numeric.mean()).abs() / numeric.std()
    return z_scores > n_std
