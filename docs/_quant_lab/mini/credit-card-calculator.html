What it does

Generates a month-by-month payoff schedule for credit card debt, showing how each payment splits between principal and interest. Compares different monthly payment amounts to reveal the true cost of carrying a balance.

The math

The fixed monthly payment that retires a balance in n months uses the standard annuity formula:

PMT = balance * r * (1 + r)^n / ((1 + r)^n - 1)
where r = annual_rate / 12

If you only pay the minimum, most of your payment goes to interest and the balance barely shrinks.

When to use it

Use this to compare payoff strategies. Doubling your payment from $150 to $300 on a $5,000 balance at 22% APR cuts total interest by more than half. At the minimum payment floor ($25), that same balance takes over 25 years to clear.

Core function

from dataclasses import dataclass


@dataclass
class PayoffRow:
    month: int
    payment: float
    principal: float
    interest: float
    balance: float


def credit_card_schedule(
    balance: float,
    annual_rate: float,
    monthly_payment: float,
    max_months: int = 600,
) -> list[PayoffRow]:
    """Generate a month-by-month payoff schedule."""
    r = annual_rate / 12
    schedule: list[PayoffRow] = []

    for month in range(1, max_months + 1):
        interest = round(balance * r, 2)
        payment = min(monthly_payment, balance + interest)
        principal = round(payment - interest, 2)
        balance = round(balance - principal, 2)
        schedule.append(PayoffRow(month, payment, principal, interest, balance))
        if balance <= 0:
            break

    return schedule

Quick usage

schedule = credit_card_schedule(
    balance=5_000.00,
    annual_rate=0.22,
    monthly_payment=200.00,
)
total_interest = sum(r.interest for r in schedule)
print(f"Months to payoff: {len(schedule)}")
print(f"Total interest:   ${total_interest:,.2f}")
