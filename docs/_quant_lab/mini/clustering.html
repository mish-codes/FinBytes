What it does

Groups customers into segments using K-Means clustering on income, spending, and credit score features. Includes the elbow method for choosing k, silhouette scoring, 2D/3D scatter plots, and cluster profile tables that map to business actions.

The math

K-Means minimizes within-cluster sum of squares (inertia). Features must be scaled first since K-Means uses Euclidean distance. The elbow method plots inertia vs k -- the "elbow" is where adding clusters gives diminishing returns. Silhouette score (range -1 to 1) measures cluster separation quality.

When to use it

Use this to drive targeted products: high-income savers get investment offers, budget-conscious customers get financial literacy tools. Clustering is the starting point -- the real value comes from connecting segments to product strategies and measuring conversion rates.

Core function

import numpy as np
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score


def cluster_customers(
    df: pd.DataFrame,
    features: list[str],
    n_clusters: int = 4,
) -> tuple[pd.DataFrame, float]:
    """Run K-Means clustering and return labeled data + silhouette score."""
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(df[features])

    kmeans = KMeans(n_clusters=n_clusters, random_state=42, n_init=10)
    df = df.copy()
    df["cluster"] = kmeans.fit_predict(X_scaled)

    score = silhouette_score(X_scaled, df["cluster"])
    return df, score


def find_optimal_k(df: pd.DataFrame, features: list[str], k_range=range(2, 10)):
    """Run elbow method: return inertias for each k."""
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(df[features])
    inertias = []
    for k in k_range:
        km = KMeans(n_clusters=k, random_state=42, n_init=10)
        km.fit(X_scaled)
        inertias.append(km.inertia_)
    return list(k_range), inertias

Quick usage

features = ["income", "monthly_spending", "credit_score"]
df_labeled, score = cluster_customers(df, features, n_clusters=4)
print(f"Silhouette score: {score:.3f}")
print(df_labeled.groupby("cluster")[features].mean().round(0))
