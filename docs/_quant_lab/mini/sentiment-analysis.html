What it does

Quantifies the emotional tone of financial headlines using two NLP tools: TextBlob (pattern-based polarity) and VADER (valence-aware, better for short texts). Compares their agreement rate and visualizes sentiment over time with rolling averages.

The math

VADER's compound score ranges from -1 (most negative) to +1 (most positive). Common thresholds: compound >= 0.05 is positive, <= -0.05 is negative. TextBlob returns polarity (-1 to 1) and subjectivity (0 to 1). Both are lexicon-based -- they look up words and average scores.

When to use it

Use this as a first pass on news sentiment. A negative sentiment shift in coverage often precedes price drops. However, general-purpose tools misread financial jargon ("volatile" is neutral in finance). For production, consider FinBERT or a transformer fine-tuned on financial text.

Core function

from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
from textblob import TextBlob
import pandas as pd

analyzer = SentimentIntensityAnalyzer()


def vader_sentiment(text: str) -> dict:
    """Return VADER compound score and component scores."""
    scores = analyzer.polarity_scores(text)
    return {
        "compound": scores["compound"],
        "positive": scores["pos"],
        "negative": scores["neg"],
        "neutral": scores["neu"],
    }


def textblob_sentiment(text: str) -> dict:
    """Return polarity (-1 to 1) and subjectivity (0 to 1)."""
    blob = TextBlob(text)
    return {
        "polarity": round(blob.sentiment.polarity, 4),
        "subjectivity": round(blob.sentiment.subjectivity, 4),
    }

Quick usage

headline = "Tech stocks surge on strong earnings beat"
print(f"VADER:    {vader_sentiment(headline)['compound']:.3f}")
print(f"TextBlob: {textblob_sentiment(headline)['polarity']:.3f}")
