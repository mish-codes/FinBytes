What is the AWS & Credit Risk Arc?

Twelve exercises that build a complete bond and credit risk analytics pipeline on AWS. Starting from raw Treasury yield data on S3, the arc constructs par curves, bootstraps spot rates, fits Nelson-Siegel models, prices bonds with OAS and Z-spread, computes default probabilities and expected losses, streams credit spreads via WebSockets, and culminates in a Monte Carlo Credit VaR simulation with correlated spread movements.

Pipeline Flow

  1. Ingest — FRED Treasury data → S3 (versioned, partitioned by date)
  2. Store — RDS PostgreSQL with Alembic-managed 6-table yield schema
  3. Compute — Lambda functions for spot curve bootstrapping, forward rates, Nelson-Siegel fitting
  4. Deploy — Terraform IaC, GitHub Actions CI/CD with OIDC (no stored AWS keys)
  5. Message — SQS/SNS for credit spread event processing, WebSockets for real-time streaming
  6. Cache — ElastiCache Redis for bond price and yield curve caching
  7. Monitor — CloudWatch dashboards and alarms
  8. Analyse — OAS/Z-spread via binomial trees, default probability term structures, Monte Carlo Credit VaR
